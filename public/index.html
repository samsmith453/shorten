URL Shortener Microservice

  • You can pass an URL as a paramter and you will receive a shortened version
  • You can visit the shortened URL and you will be sent to the target URL

Example input

https://shortarse.herokuapp.com/http://www.google.com

Example Output

{"longUrl":"http://www.google.com","shortUrl":"https://shortarse.herokuapp.com/1573"}

If you visit the 'shortUrl' site

You will be sent to http://www.google.com
